# CloudPDV
API de Integração de Ponto de Vendas

## Foram Utilizados no Projeto 
 - Spring Boot
 - Spring Data JPA
 - Flyway Migration (Versionamento de Base de Dados)
 - JWT (Autenticação)
 - Hikari(Pool de Conexões)
 - Lombok
 - Swagger (Documentação)
 - Junit (Teste Unitario)
 - Integracao AWS S3
